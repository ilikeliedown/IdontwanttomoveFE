## 나 기숙사에서 움직이기 싫어.
이 프로젝트는 교내 기숙사 (319호)에서 불편한 점을 개선하기 위해 개발하게 된 프로젝트입니다.

불편점은 아래와 같았습니다. (힘들어서 쉬고 있을 때를 전제로 합니다.)
> 1. 문을 열어달라고 하면 문을 열어줘야한다.
> 2. 소등시간이 되면 방의 불과 베란다 불을 꺼야한다.
> 3. 자기 전 콘센트 전원을 꺼야한다.
> 4. 화장실에 습도가 높다.

4가지의 불편점을 해결하기 위해서 서비스를 개발하게 되었습니다.

## 프로젝트 시작하기
프로젝트는 Svelte Kit으로 개발 중입니다.
> npm install

> npm run dev
